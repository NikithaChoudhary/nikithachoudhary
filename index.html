Projects

Constituency Parser

Built a Constituency parser from ATIS portion of PennTreebank. The program learns the context free grammar from the input trees and extends Viterbi with head lexicalization and head binarization to parse effectively with context awareness.

NER Tagger

Built a supervised named entity recognizer for Twitter dataset. The program performs feature engineering, and the conditional random field implementation uses Viterbi algorithm to do efficient sequence tagging.
